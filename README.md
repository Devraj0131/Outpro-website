# Outpro India Corporate Website

A full-stack corporate website built using HTML, CSS, JavaScript, Node.js, Express.js, and MongoDB Atlas.

## Features
- Responsive design
- Contact form with MongoDB integration
- Admin dashboard
- Mobile responsive navbar
- Cloud deployment using Render

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MongoDB Atlas
- Render
